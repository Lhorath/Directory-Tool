# Project hub (Directory Tool)

Single-file **PHP** “project hub” that showcases apps and utilities. SEO metadata, Open Graph, and canonical URLs are built from the current request. `SITE_ORIGIN` and `BASE_URL` in `index.php` should match your deployment.

## Live demo

http://directory-tool.dackdns.ddns.net/

## Requirements

- PHP-capable host.
