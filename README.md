# Directory Tool — Project Hub

> Single-file PHP project hub that showcases apps and utilities with SEO metadata, Open Graph tags, and canonical URLs built from the current request.

## Live Demo

[directory-tool.dackdns.ddns.net](http://directory-tool.dackdns.ddns.net/)

## Configuration

In `index.php`, set `SITE_ORIGIN` and `BASE_URL` to match your deployment URL.

## Requirements

- PHP-capable web host

## Usage

1. Deploy `index.php` (and any supporting files) to your web host.
2. Update `SITE_ORIGIN` and `BASE_URL` constants.
3. Add or edit project entries in the hub as needed.
